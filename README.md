## 🧩 功能一览

### 🔁 智能连点
- 支持 **鼠标左键 / 右键** 切换  
- 间隔范围 **1 毫秒 ～ 60 秒**，满足超高速连点与慢速模拟  
- **锁定模式**：勾选后仅通过全局热键控制，防止误触  
- 状态实时显示，连点中/已停止一目了然  

### 🎥 操作录制与回放
- 录制 **鼠标移动、点击、键盘输入** 的所有动作  
- 自动过滤微小抖动，文件体积更小  
- 保存为专用 `.rec` 文件，随时加载  
- **一键回放**，完美复现录制的操作序列  
- 回放过程支持随时中断  

### ⌨️ 全局热键
- 自定义启动热键（例如 `Ctrl+F1`、`Alt+Z`）  
- 弹出式捕获窗口，按下想要的组合键即可绑定  
- 配合“锁定”功能，让连点随时听你指挥  

### 🖥️ 现代中文界面
- 基于 Qt 打造，原生 Windows 界面，稳定流畅  
- 所有按钮、提示均为中文，无需语言障碍  
- 布局清晰，设置区、控制区、录制区一目了然

## 🚀 快速上手

1. **选择连点按钮**（左键/右键）并设置间隔  
2. **设置启动热键**（或直接点击“开始连点”）  
3. 如需热键控制，勾选 **“锁定”**，按热键启停  
4. 录制操作：点击“开始录制” → 执行操作 → “停止录制” → “保存录制”  
5. 回放时，加载录制文件，点击“播放”即可
